为决策而设计

不只是看汇率,更帮你判断何时换汇。

📊

实时牌价

现汇 / 现钞买入卖出价、中行折算价,直接解析自中国银行官网,不经第三方。

📈

走势 · 便宜指数

每张卡片内嵌 30 / 90 / 365 天迷你走势,并标出当前价在区间中的位置——一眼看出"现在贵不贵"。

🧮

货币换算器

自动选对买入/卖出、现汇/现钞,支持任意两种外币交叉换算,并显示买卖点差。

🔔

智能提醒

阈值突破、每日定点摘要、大幅波动、N 日新高/新低——抓住换汇时机。

🎯

图标角标

把你最关心的汇率钉在工具栏 / 菜单栏图标上,按当日涨跌红涨绿跌,不打开也知道行情。

🌗

多端 · 暗色 · 多语言

Chrome / Edge 扩展与 macOS / Windows 桌面应用;自动跟随系统暗色;10 种界面语言。
